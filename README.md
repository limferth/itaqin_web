# ITAQIN Web

Sitio corporativo multipágina para ITAQIN, una fintech enfocada en financiamiento, educación, consultoría, software y comunidad para MYPES.

## Estructura

- `index.html`: home corporativa.
- `productos.html` y `productos/*.html`: portafolio financiero y páginas individuales.
- `servicios.html`, `educacion-financiera.html`, `consultoria.html`, `software-empresarial.html`: servicios de crecimiento.
- `casos-exito.html`, `equipo.html`, `blog.html`, `contacto.html`, `faq.html`, `agenda-reunion.html`: confianza, contenido y conversión.
- `terminos.html`, `privacidad.html`: políticas.
- `css/styles.css`: sistema visual y responsive mobile-first.
- `js/main.js`: navegación mobile, animaciones, acordeones, formularios, contadores, modal, toast y botón volver arriba.

## Uso

Abrir `index.html` en el navegador o servir la carpeta raíz con cualquier servidor estático.
